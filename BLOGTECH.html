something interessant that I have discovered:

  • if I try to use a style entry directly in the style sheet like

    ul, #htreedemo { list-style-type: none; }
    #htreedemo { margin: 0; padding: 0; }

    Obviously it removes the list dots everywhere. OK, it's logic

  • So the correct approach is to do setAttribute on every tree ul elements.

    This way it'll concern only this demo items.
